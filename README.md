
# Amazon Shopping's Review Analysis Dashboard (Sentimental Analysis)

This project is an end-to-end Data Analytics solution built using Python and Power BI to analyze customer reviews from Amazon products.

The dataset was collected from Kaggle and processed using Natural Language Processing (NLP) techniques to classify customer reviews into Positive, Negative, and Neutral sentiments. The cleaned data was then visualized through an interactive Power BI dashboard to uncover customer opinions and review trends.

---
## Features

- Analyze customer sentiments from Amazon product reviews.
- Identify the distribution of Positive, Negative, and Neutral reviews.
- Build an interactive dashboard for business decision-making.
- Demonstrate an end-to-end analytics workflow using Python and Power BI.

## Key Insights 

- Around 75% of customer reviews are positive.
- Negative reviews account for nearly 19% of all reviews.
- Neutral reviews contribute approximately 6%.
- Most customers express overall satisfaction with their purchases.
- Sentiment analysis helps identify areas where products or services can be improved.
